后台服务接口 后台服务接口主要是支撑系统的一些后台服务。
如果要实现某一个后台服务，则需要实现后台服务结构。
自定义的后台服务实现的步骤：
1.需要继承 t9.core.autorun.T9AutoRun类 重写doTask() 方法
注意：通过Connection conn = getRequestDbConn().getSysDbConn();能得到数据库的连接
2.在系统设置中配置自定义的系统服务
 java代码:
public class T9WebmailAutoService extends T9AutoRun {
  public void doTask() {
    System.out.println("T9WebmailAutoService doTask Run" + T9Utility.getCurDateTimeStr());
    try {
      Connection conn = getRequestDbConn().getSysDbConn();
    } catch (Exception e) {
    }
    System.out.println("T9WebmailAutoService doTask Run END " + T9Utility.getCurDateTimeStr());
  }
}
