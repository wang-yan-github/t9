适用情景描述 特别强调 代码
Excel导出 T9JExcelUtil.writeExc(OutputStream ops,ArrayList <T9DbRecord > dataArray ); 此方法用于excel的导出 java代码：
ArrayList<T9DbRecord> dbL = new ArrayList<T9DbRecord >();
for (int i = 0; i < 10; i++) {
  T9DbRecord dbrec = new T9DbRecord();
  dbrec.addField("标题1", "值1-" + i);
  dbrec.addField("标题2", "值2-" + i);
  dbL.add(dbrec);
}
T9JExcelUtil.writeExc(ops,dbL );
Csv导出 T9CSVUtil.CVSWrite(OutputStream ops,ArrayList <T9DbRecord > dataArray ); 此方法用于csv的导出 java代码：
ArrayList<T9DbRecord> dbL = new ArrayList<T9DbRecord >();
for (int i = 0; i < 10; i++) {
  T9DbRecord dbrec = new T9DbRecord();
  dbrec.addField("标题1", "值1-" + i);
  dbrec.addField("标题2", "值2-" + i);
  dbL.add(dbrec);
}
T9CSVUtil.CVSWrite(ops,dbL );
Csv导入 T9CSVUtil.CVSReader(InputStream is,String charset);
此方法将csv文件解析为T9DbRecord的一个列表，用于导入操作 		java代码：
ArrayList<T9DbRecord> drl = T9CSVUtil.CVSReader(is,"gbk");
