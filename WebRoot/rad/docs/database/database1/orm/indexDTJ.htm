StringBuffer StringBuffer toJson(Connection conn,String tableNo,Integer pageNum,Integer pageRows,String[] filters)
此方法处理分页的数据
  • 参数说明
    • conn : 数据库连接
    • tableNo : 数据表编码
    • pageNum ：分页页码
    • pageRows ： 每页显示条数
    • filters ： where查询条件:所对应的值如为 SEQ_ID = 1 则String的值为"SEQ_ID=1",如为 NAME = "SS" 则String的值为"NAME='SS'"
    • 主要实现无对象的数据与数据库的关联。主要用于数据的展示
    能够提供小编码数据的展示功能，提供数据库级别的分页功能
