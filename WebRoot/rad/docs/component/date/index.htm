自定义时间日期格式 
样式表：[cssPath]/cmp/Calendar.css
Js文件：[contextPath]/core/js/cmp/Calendarfy.js
//配置参数
  var beginParameters = {
      //时间接收input框的id
      inputId:'timeInput',
      //isHaveTime是否有时间

      property:{isHaveTime:true,format:'yy年M月d日'}
      ,bindToBtn:'imgBtn'
  };
  new Calendar(beginParameters);

加入property参数,并设置其中的format
format:带时间时默认为"yyyy-MM-dd hh:mm:ss",不带时间:"yyyy-MM-dd"
/rad/docs/component/date/calendar4.jsp
时间日期选择其它配置 
样式表：[cssPath]/cmp/Calendar.css
Js文件：[contextPath]/core/js/cmp/Calendarfy.js
//配置参数
  var beginParameters = {
      //时间接收input框的id
      inputId:'timeInput',
      //isHaveTime是否有时间

      property:{yearRange:[2005,2015]
                  ,month:['Jan','Feb','Mar','Apr','May','Jun','Jul','Aug','Sept','Oct','Nov','Dec']
                  ,week:['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat']}
      ,bindToBtn:'imgBtn'
  };
  new Calendar(beginParameters);

加入property参数
设置其中的yearRange,可以设置日期选择的范围如这时只能选择2005年到2010年

设置其中的month,可以自定义显示月的名称默认为
  ['一月','二月','三月','四月','五月','六月','七月','八月'
        ,'九月','十月','十一月','十二月']
设置其中的week,可以自定义显示周的名称默认为
  ['日','一','二','三','四','五','六']
/rad/docs/component/date/calendar5.jsp
引入文件及参数说明 
样式表：[cssPath]/cmp/Calendar.css
Js文件：[contextPath]/core/js/cmp/Calendarfy.js
接口函数：Calendar
接口参数:{bindToBtn,inputId,property:{week,yearRange,month,format,ishaveTime}}
    bindToBtn 字符串 绑定的按钮id,不指定日历会默认显示在输入框下面
    inputId 字符串 日历输入框的id
    property 对象 主要设置日历的一些属性

      week 字符串数组 指定日历显示星期的格式

      month 字符串数组 指定日历显示月份的格式

      yearRange 整型数组 指定年的显示范围
      format 字符串 日期的输出格式

      ishaveTime boolean 是否显示时间\
