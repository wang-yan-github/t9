/rad/docs/component/asstinput/assistInput.jsp
引入文件及参数介绍 
Js文件：[contextPath]/core/js/cmp/AssistInput1.0.js
样式表：[cssPath]/cmp/AssistInput.css
js对象:AssistInuput
构造函数参数:{bindToId,requestUrl,showLength,func}
    bindToId 字符串  所要邦定的输入控件
    requestUrl 字符串 请求路径
    showLength int 最多显示的条数
    func 函数 超出最多显示条数时点击'...'所执行的函数

后台提供的数据结构：
    {rtState:"0", rtMsrg:"成功返回!"
        , rtData:{count:,lis:
                     [{id:,string:},....]}}
    count int 总条数,不是显示的条数

    lis 对象数组 每条的数据其中,id废弃了,string每一条显示的label
