数据列表翻页导航


作者: 陈懿 创建日期: 2009-09-23 版本: 1.0
适用情景描述 特别强调 代码
数据列表无分页

JS文件：rad/grid/grid.js
css文件:rad/grid/grid.css
构造函数：T9Grid(hdarray,url,isSynchronous,numOfPage);
hdarray:列表的头信息
url：数据请求的地址
isSynchronous：是否异步（暂未实现） 值为null就行。
numOfPage：每页显示的数据条数
hdarray的结构：var hd =[
[
{header:"hID",name:"id",width:"10%",hidden:true},
{header:"hAge",name:"age",width:"20%"},
{header:"hanme",name:"name",width:"40%"}
],
{
header:"操作",
oprates:[
new T9Oprate('增加',true,function(record,index){alert(record.getField('name').value+":"+index)}),
new T9Oprate('oprate4',false,function(){alert("oprate4")}),
new T9Oprate('删除',true,function(){alert("oprate2")})
]
}
];
hdarray说明：name 数据中是所对应的每列的列明，header是name的别名（用于显示），width 每列的款度，hidden为TRUE则表示此为隐藏字段
操作区： T9Oprate(name,isDefault,oprate) name操作名称，isDefault是否为默认操作，非默认操作将显示在select中，同时请
最多设置两个默认操作，oprate操作所要执行的函数，此函数默认大有两个参数record 当前行的纪录，index当前行号。
grid.rendTo（dom）；将grid加载到指定的html容器中，dom为所要容纳grid的html组件的ID
grid.reShow（url）；重新加载数据 url为数据获得的地址
grid.records : 本次请求的数据 Array[] record数据.

 
    		调用grid的rendTo方法是必须在页面元素加载完之后。

    		grid.records只在数据加载完成后才存在，在rendTo或者

    		reShow方法之后才有效
