 Prototype框架：js/prototype.js
 接口函数：MyTreeInit
接口函数参数:rootNodeId,requestUrl,treeStructure,isOnceLoad,checkboxPara,linkPara
       rootNodeId:所依附的节点

       requestUrl:请求服务器的url地址
       treeStructure：树的结构

       isOnceLoad：属性为true或false，true为一次性加载，false非一次性加载

       checkboxPara：复选框参数
       linkPara：链接参数

    	
       treeStructure:包含两个参数：第一个参数说明是否isNoTree，true为是。第二个参数说明如果是idNoTree，需
指定规则regular
       checkboxPara：表明是否有复选框。如果有，选中复选框的回调函数，回调函数的参数值是test，选中
复选框后的撤销函数的参数是test1
       linkPara：表明是否有链接。如果有，指定链接地址和链接页面在浏览器中的打开方式
